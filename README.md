# 🚗 Real-Time Vehicle Telemetry Analytics Dashboard

## 📖 Overview

This project simulates **real-time vehicle telemetry data**, automates the ETL process using Python, stores data in PostgreSQL, and visualizes fleet performance using an interactive Power BI dashboard.

The project demonstrates practical data engineering and business intelligence skills through an end-to-end analytics pipeline.

---

## 🎯 Business Problem

Fleet management companies collect large volumes of vehicle telemetry data. Monitoring vehicle performance manually is inefficient and can lead to delayed maintenance decisions.

This project automates data collection, storage, analysis, and visualization to support data-driven fleet management.
